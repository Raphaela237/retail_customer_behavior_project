# Analyse du comportement d’achat client – Data Analytics End-to-End

## Contexte du projet
Ce projet a pour objectif d’analyser des données transactionnelles clients afin de mieux comprendre les comportements d’achat et d’identifier des leviers d’action concrets pour les équipes commerciales et marketing. L’approche couvre l’ensemble du cycle de vie de la donnée, de l’exploration et la fiabilisation des données brutes jusqu’à leur exploitation dans un outil de pilotage décisionnel.

---

## Objectifs
- Identifier les segments de clients les plus contributeurs au chiffre d’affaires
- Analyser la performance des catégories de produits
- Construire des indicateurs fiables pour le pilotage commercial
- Préparer des données exploitables dans un dashboard Power BI

---

## Stack technique
- Python : pandas, numpy, matplotlib, SQLAlchemy
- Base de données : PostgreSQL (hébergée sur Neon)
- Visualisation : Power BI
- Versioning : Git / GitHub

---

## Installation et utilisation

**Cloner le projet**  
Commencer par cloner le projet depuis GitHub et se déplacer dans le dossier du projet.

**Créer et activer un environnement virtuel et installer les dépendances**  
Créer un environnement virtuel pour isoler les bibliothèques nécessaires. Une fois activé, installer toutes les dépendances requises pour exécuter le projet.

**Configurer les variables de connexion à la base de données**  
Renseigner les informations de connexion à la base de données dans un fichier `.env` à la racine du projet.

**Lancer le notebook Jupyter**  
Lancer Jupyter Notebook, sélectionner le kernel correspondant à l’environnement virtuel, et exécuter les cellules pour analyser les données et vérifier la connexion à la base de données.
